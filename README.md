# 🧮 Calculatrice Console en Java

## 📝 Description du Projet

Ce projet a été développé dans le but d'apprendre et de maîtriser les concepts fondamentaux de la programmation en Java. Il s'agit d'une calculatrice en mode console 🖥️ qui permet à l'utilisateur d'effectuer des opérations mathématiques de base telles que l'addition, la soustraction, la multiplication et la division.

## ✨ Fonctionnalités

- **Saisie de Nombres** 🔢: Permet à l'utilisateur de saisir deux nombres pour effectuer des opérations.
- **Opérations Mathématiques** ➕➖✖️➗: Effectue des opérations de base telles que l'addition (`+`), la soustraction (`-`), la multiplication (`*`) et la division (`/`).
- **Gestion des Erreurs** ⚠️: Comprend une gestion des erreurs, comme la division par zéro.
- **Boucle Continue** 🔁: Permet à l'utilisateur d'effectuer plusieurs calculs sans redémarrer le programme.

## 🛠 Aspects Techniques

- **Langage de Programmation** 🌐: Le projet est entièrement écrit en Java.
- **Utilisation des Énumérations** 🔢: Utilisation d'une énumération pour gérer les différentes opérations mathématiques.
- **Interfaces Fonctionnelles** 🔄: Emploi des interfaces fonctionnelles, notamment `BiFunction`, pour associer chaque opération mathématique à son comportement fonctionnel correspondant.
- **Gestion des Exceptions** 🚫: Implémentation d'une gestion des exceptions pour traiter les cas spéciaux comme la division par zéro.
- **Boucles et Conditions** ➿: Utilisation de boucles `while` et `do-while` pour la saisie des données et la navigation dans le menu de la calculatrice.

## 🎯 Pourquoi ce Projet ?

Le but principal de ce projet était de renforcer ma compréhension de Java en tant que langage de programmation orienté objet. En travaillant sur ce projet, j'ai pu approfondir mes connaissances des concepts clés de Java, tels que les énumérations, les interfaces fonctionnelles, la gestion des exceptions, et la structure de contrôle des flux.
